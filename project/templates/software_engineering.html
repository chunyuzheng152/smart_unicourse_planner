Software Engineering

Software Engineering focuses on designing, building, testing, and maintaining software systems.
It combines programming skills with project management and software development methods.

Courses

Software Engineering focuses on the full software development process. Students learn how to design, build, test, maintain, and improve software systems for real users and organisations.

Course Area What Students Learn Why It Matters
Programming Writing code using languages such as Python, Java, C, or JavaScript. Programming is the basic tool for creating software systems.
Software Design System design, object-oriented design, architecture, and design patterns. Good design makes software easier to build, maintain, and extend.
Software Testing Debugging, unit testing, quality assurance, and test planning. Testing helps reduce errors and improve software reliability.
Databases Database design, SQL, data storage, and data management. Most software applications need to store and manage data.
Web and Application Development Building websites, mobile apps, APIs, and user-facing systems. This helps students create practical software products.
Software Project Management Teamwork, agile development, version control, documentation, and project planning. Real software projects need coordination, communication, and clear development processes.
For official course information, please check the UWA Software Engineering course page .

Skills

Software Engineering students develop strong programming skills, but they also learn how to work in teams and manage complete software projects.

Technical Skills
  • Programming with common software development languages.
  • Designing software systems and application structures.
  • Using databases and APIs in software projects.
  • Testing, debugging, and improving code quality.
  • Using Git and version control in team projects.
  • Building web, mobile, or desktop applications.
Transferable Skills
  • Teamwork in software development projects.
  • Clear communication with technical and non-technical users.
  • Project planning and time management.
  • Problem solving and logical thinking.
  • Writing documentation for software systems.
  • Understanding user needs and system requirements.
Programming Software Design Testing Git Project Management Teamwork

Careers

Software Engineering graduates can work in technology companies, banks, consulting firms, government departments, startups, and organisations that build or maintain digital systems.

Software Engineer

Software engineers design, develop, test, and maintain software systems.

Application Developer

Application developers build desktop, mobile, or web applications for users.

Web Developer

Web developers create websites and online systems using front-end and back-end technologies.

Software Tester

Software testers check software quality, find bugs, and help improve reliability.

Systems Analyst

Systems analysts study user needs and help design software solutions for organisations.

DevOps Assistant

DevOps assistants support deployment, automation, version control, and system operations.

Fees

Tuition fees for Software Engineering may vary depending on the university, student status, study load, and selected units. Domestic and international students may have different fee structures.

Domestic Students

Domestic students may pay fees based on their course structure, Commonwealth-supported place status, and selected units. The final cost can change depending on annual study load.

International Students

International students usually pay annual tuition fees. The amount can vary by course, year of study, and enrolment load.

Tuition fees may change each year. Please use the UWA Tuition Fee Calculator to check the latest estimated cost.

Scholarship Type Description

Admission Requirements

Admission requirements for Software Engineering depend on the university, course structure, and the applicant's background. Students usually need to meet academic entry requirements and English language requirements.

Academic Background
  • Completion of high school or equivalent qualification.
  • Meeting the required ATAR or equivalent entry score.
  • Mathematics background is often recommended or required.
  • Some engineering courses may require specific science or mathematics subjects.
English Requirement
  • International students usually need to meet English language requirements.
  • Accepted tests may include IELTS, TOEFL, or equivalent tests.
  • The exact score depends on the university and course level.
Pathway Options

Students who do not meet direct entry requirements may consider foundation programs, diploma pathways, internal transfer options, or related computing and engineering pathways, depending on the university.

Entry requirements may depend on the course and applicant background. Please check the UWA Entry Standards page for official entry information.
